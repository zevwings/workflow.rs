# 覆盖率测试指南

> 本文档介绍测试覆盖率的检查和提升方法。

---

## 覆盖率工具

### 安装

```bash
cargo install cargo-tarpaulin
```

### 生成覆盖率报告

```bash
# HTML格式
make coverage

# Lcov格式（CI）
make coverage-ci
```

## 覆盖率目标

- **总体覆盖率**：> 80%
- **关键业务逻辑**：> 90%
- **工具函数**：> 70%

## 覆盖率提升技巧

1. 识别低覆盖率区域
2. 补充边界测试
3. 添加错误处理测试
4. 使用参数化测试
