# Git 分支分离计划

## 目标
将当前 `refactor/ui-framework-ratatui` 分支的修改分为两个独立分支：
1. **ratatui 分支**：只包含 ratatui/crossterm UI 框架迁移
2. **other 分支**：包含其他功能修改

## 执行步骤

### 方案 A：使用 git stash 分离（推荐）

```bash
# 1. 确保当前在 refactor/ui-framework-ratatui 分支
git checkout refactor/ui-framework-ratatui

# 2. 创建 ratatui 专用分支（从当前状态）
git checkout -b refactor/ratatui-only

# 3. 暂存所有修改
git add -A
git stash

# 4. 重置到 master，只保留 ratatui 相关文件
git reset --hard master

# 5. 恢复修改，但只添加 ratatui 相关文件
git stash pop

# 6. 只添加 ratatui 相关文件
git add Cargo.toml Cargo.lock
git add src/lib/base/ui/
git add src/lib/logging/
git add src/lib/base/util/confirm.rs
git add src/lib/base/util/logger.rs  # 删除的文件
git add src/lib/base/util/mod.rs
git add src/lib/base/mod.rs
git add src/lib.rs
git add src/bin/workflow.rs
git add src/commands/**/*.rs  # 所有命令文件
git add tests/ui_*.rs

# 7. 提交 ratatui 分支
git commit -m "refactor: migrate UI framework to ratatui + crossterm

- Add ratatui and crossterm dependencies
- Replace dialoguer with ratatui dialogs
- Replace indicatif with ratatui progress bars
- Replace colored with ratatui styles
- Add new UI components module
- Add logging module with ratatui integration
- Update all commands to use ratatui dialogs"

# 8. 创建其他修改分支
git checkout refactor/ui-framework-ratatui
git checkout -b refactor/other-changes

# 9. 重置并只添加其他文件
git reset --hard master
git stash pop

# 10. 只添加其他修改文件
git add Makefile
git add docs/
git add src/lib/base/http/retry.rs
git add src/lib/jira/status.rs
git add src/commands/lifecycle/update.rs
git add tests/jira_helpers.rs tests/logger.rs
git add tests/commands/ tests/common/ tests/*.rs

# 11. 提交其他修改分支
git commit -m "refactor: other improvements and updates

- Update Makefile
- Update documentation
- Improve HTTP retry logic
- Update JIRA status handling
- Add/update tests"

# 12. 清理
git stash drop
```

### 方案 B：使用交互式选择（更精确）

```bash
# 1. 创建 ratatui 分支
git checkout refactor/ui-framework-ratatui
git checkout -b refactor/ratatui-only

# 2. 重置到 master
git reset --hard master

# 3. 使用 git checkout 从原分支恢复特定文件
git checkout refactor/ui-framework-ratatui -- Cargo.toml Cargo.lock
git checkout refactor/ui-framework-ratatui -- src/lib/base/ui/
git checkout refactor/ui-framework-ratatui -- src/lib/logging/
# ... 继续添加其他 ratatui 相关文件

# 4. 提交
git commit -m "refactor: migrate UI framework to ratatui"

# 5. 创建其他修改分支
git checkout refactor/ui-framework-ratatui
git checkout -b refactor/other-changes
git reset --hard master

# 6. 恢复其他文件
git checkout refactor/ui-framework-ratatui -- Makefile
git checkout refactor/ui-framework-ratatui -- docs/
# ... 继续添加其他文件

# 7. 提交
git commit -m "refactor: other improvements"
```

## 注意事项

1. **依赖冲突**：如果两个分支都修改了 `Cargo.toml`，需要确保合并时正确处理
2. **测试文件**：某些测试可能同时依赖 ratatui 和其他功能，需要仔细分类
3. **文档文件**：`docs/todo/` 可能包含混合内容，需要检查
4. **合并顺序**：建议先合并 ratatui 分支，再合并其他修改分支

## 验证

分离后，验证每个分支：
- `refactor/ratatui-only`：应该只包含 UI 框架相关修改
- `refactor/other-changes`：应该只包含其他功能修改
- 两个分支都应该能独立编译和测试
